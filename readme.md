## О проекте

Верстка многостраничного сайта и настройка JavaScript функционала для компании АгроМир

**Реализованный функционал:**
* Анимации
* Слайдеры (использована библиотека 'slick')
* Пагинация
* Фильтры каталога
* Модальные окна
